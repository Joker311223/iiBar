iibar:// 自动化

可从终端、快捷指令、Raycast、Alfred 或任意能打开 URL 的应用控制 iiBar。

动作

URL作用
iibar://toggle-hidden显示或隐藏“隐藏”区域
iibar://toggle-always-hidden显示或隐藏“始终隐藏”区域
iibar://search打开项目搜索
iibar://toggle-iibarbar切换 iiBar Bar
iibar://toggle-application-menus切换应用菜单
iibar://open-settings打开设置窗口
open "iibar://search"

修改设置

使用 set 设置值，或使用 toggle 翻转布尔值：

open "iibar://set?key=autoRehide&value=true"
open "iibar://toggle?key=hideApplicationMenus"

部分显示器级设置支持附加 display 参数。具体可用键以源码中的 SettingsURIHandler 为准。

读取设置

get 需要提供回调 URL。iiBar 会把 JSON 结果编码到回调的 data 查询参数中：

open "iibar://get?key=autoRehide&callback=myapp://response"
调试工具

仓库内的 iiBarCtl 可以发送请求并展示回调，适合验证集成。

授权与安全

外部应用首次读写设置时需要用户确认。应用也可以主动请求授权：

open "iibar://authorize"

只有获得许可的调用方才能读写设置；普通的显示、搜索和打开设置动作不涉及设置读取。
